# Programa ACAR Sabatino - Panel de Control v2

Panel de control administrativo completo para el Programa ACAR Sabatino. Sistema de gestion de asistencia, programas, institutos, materias y reportes.

---

## Modulos

- **Dashboard** - Panel principal con QR, tabla de asistencia en tiempo real, stats y exportacion Excel
- **Programas** - Gestion de programas academicos con pensum
- **Institutos** - Gestion de institutos con servicios
- **Materias** - Gestion de materias con temario
- **Estadisticas** - Graficas de asistencia con recharts
- **Reportes** - Exportacion XLSX/PDF con filtros
- **Ajustes** - Perfil, notificaciones y configuracion

---

## Tecnologias

- React + Vite
- Tailwind CSS v4 con glass morphism
- motion/react v12 para animaciones
- recharts para graficas
- Firebase Realtime Database
- ExcelJS para exportacion
